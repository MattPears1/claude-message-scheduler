CLAUDE MESSAGE SCHEDULER
========================

FIRST TIME SETUP:
1. Double-click ENABLE_AUTOSTART.bat (starts with Windows)
2. That's it! It will always be running.

TO USE:
- Press F9 to schedule messages
- Press Shift+F9 to view/copy/cancel messages

MANUAL START:
- Double-click START.bat (if not using autostart)

AUTOSTART OPTIONS:
- ENABLE_AUTOSTART.bat = Quick setup (recommended)
- INSTALL_AUTOSTART.bat = Choose install method
- REMOVE_AUTOSTART.bat = Stop autostart

The scheduler runs in the background.
You can schedule messages for multiple windows.
Messages send automatically at their scheduled time.

Created for 24/7 Claude Code usage.
